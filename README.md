# MJEEDSTORE

مشروع تعليمي لمتجر إلكتروني متخصص في بيع اشتراكات.

تم تطويره كتطبيق عملي لمفاهيم:

- بناء واجهة مستخدم
- التعامل مع الطلبات
- ربط الموقع بقاعدة بيانات MySQL
- إنشاء لوحة تحكم إدارية
- رفع الملفات
- إدارة المنتجات والأسعار

## التقنيات المستخدمة

- PHP
- MySQL
- HTML / CSS / JavaScript

## المميزات

- عرض المنتجات من قاعدة البيانات
- سلة مشتريات
- إرسال طلب مع رفع إيصال
- تتبع حالة الطلب
- لوحة إدارة للطلبات والمنتجات
